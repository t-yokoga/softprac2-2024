    このサイトについて

    このサイトは，2024年度「ソフトウェア演習II」の演習のための Web サイトです． 本演習は，岡山県立大学 情報工学部 情報システム工学科の 2 年次生を対象として実施されます．

    更新履歴

    • 2024.9.17 サイト公開

    演習について

    演習の目的

    • 解決すべき問題をより小さな問題へと分割し、単純な機能を持つ関数の組み合わせとしてその解法を実現する能力を習得する。
    • 線形リストや二分探索木などのデータ構造や、再帰呼出を用いたアルゴリズムをプログラムとして実装する能力を習得する。

    実施形態

    • 実力確認テストとテスト復習は対面での実施となります．
      • テスト復習では答案用紙のコピーを返却するため
    • その他の演習日はハイブリッド形式 (2207演習室＋Zoom) での実施となります．体調不良等で登校が難しい方は以下の Zoom ミーティングで受講してください．オンライン受講する場合の事前連絡は不要です．
      • Zoom 会場
        • ミーティングID: 853 5819 3327
        • パスコード: 784316

    実力確認テストについて

    • 対面での受験となります．オンライン受験はできません．
      • 体調不良等で受験が難しい場合は追試験を実施しますので，事前あるいは事後に連絡してください．
    • 試験時間は 60 分です．
    • 試験範囲は C 言語全般です．参考書等の持ち込みはできません．

    生成系AIの利用について

    • 基本的に「岡山県立大学の教育における生成系AIガイドライン」に従います．
    • ソフトウェア演習II (2024年度) では，生成系AIの利用については一切の制限を設けません．利用についての申告も不要です．
