    プログラミング環境の構築について

    自宅など，2207 演習室以外の環境で作業を行うためには，別途プログラミング環境を構築する必要があります． その場合，大きく分けて

    1. コンピュータ上にコンパイラやエディタなどの開発環境をインストールする
    2. ブラウザ上で実行可能な開発環境を外部サービスとして利用する

    の 2 通りの方法があります． ここでは，コンピュータ上にコンパイラやエディタなどの開発環境をインストールする方法について説明します．

    コンピュータ上でコンパイル・実行

    Visual Studio (Windows を利用している方)

    2207 演習室の計算機には Visual Studio 2019 がインストールされているので，各自の PC に Visual Studio 2019 を導入すれば演習室とほぼ同じ環境で C 言語によるプログラミングが可能です．

    1. Visual Studio のインストール
      • Microsoft 公式の Visual Studio 2019 最新のリリースから，Visual Studio 2019 Community のインストーラをダウンロードして，実行してください．
    2. プログラムの作成
      • Visual Studio 2019 を起動する
      • 新しいプロジェクトの作成を選択
      • 空のプロジェクトを選択
      • プロジェクト名を入力
      • ソリューションエクスプローラーの ソースファイル を右クリックし，追加 > 新しい項目 を選択
        • Visual C++ から C++ ファイル (.cpp) を選択
      • エディタ上でプログラムを作成する
    3. プログラムのコンパイル・実行
      • メニューバーから デバッグ > デバッグなしで開始 を選択（あるいは Ctrl + F5）

    Xcode + Visual Studio Code (Mac OS を利用している方)

    Mac OS の場合，Xcode で GCC コンパイラをインストールできます． ここでは，Visual Studio Code を用いてプログラムを作成し，実行する方法について説明します．

    1. コンパイラのインストール
      • Xcode のコマンドラインツールを追加することで C のコンパイラ (gcc) を利用することができるようになります．
        1. ターミナル.app 上で gcc コマンドを実行
          • コマンドラインツールをインストールするかどうかを問うダイアログボックスが現れる
        2. コマンドライン・デベロッパツールをインストール
          • gcc コマンドが利用可能になる
    2. エディタのインストール
      • ここでは，コードエディタとして Visual Studio Code を使います．
        1. アプリのダウンロード
          • Microsoft の VS Code のサイトからダウンロード
          • あるいはブラウザで「VS Code」で検索
        2. アプリの起動
        3. 拡張機能の追加
          • C/C++ の拡張機能を追加しておく
    3. プログラムの作成
      1. ファイルの新規作成
        • 言語として C を選択する
      2. エディタでプログラムを記述
    4. プログラムのコンパイル・実行
      1. VS Code 内でターミナルを起動
        • メニューバーのターミナル > 新しいターミナル から起動
      2. gcc でコンパイル
        • コマンドラインで gcc example.c などと入力
      3. プログラムの実行
        • コマンドラインで /a.out などと入力
